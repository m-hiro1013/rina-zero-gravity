# 🎯 デュアルコア・ワークフロー：実戦テストプラン002

## 📋 概要
このプランは、複数のファイル作成、ライブラリのインストール、およびバックグラウンドでの動作検証を全自動で行うテストです。

## 🛠 タスクリスト
1. [ ] `pnpm init` でプロジェクトを初期化する。
2. [ ] `express` をインストールする（`pnpm add express`）。
3. [ ] `server.js` を作成する。
   - 内容：
     - `/` にアクセスすると `Welcome to Dual-Core API!` と返す。
     - `/status` にアクセスすると `{ success: true, timestamp: ... }` を返す。
     - ポートは 3000 番。
4. [ ] サーバーをバックグラウンドで起動し、実際に `/status` にリクエストを送って結果を `api_test.log` に保存する。
   - 手順：
     1. `node server.js &` (またはバックグラウンド実行)
     2. `sleep 2` (起動を待つ)
     3. `curl http://localhost:3000/status > api_test.log`
     4. ポート3000のプロセスを終了させる。

## ⚠️ 制約・指示
- ひろきくんの負担をゼロにするため、ライブラリインストール後の許可なども含め、**すべて自動**で進めてください。
- 途中でエラーが出た場合は、デバッグプロトコルに従って自律的に修正を試みてください。
- 完了したら、いつもの「通知」でお知らせしてください。
