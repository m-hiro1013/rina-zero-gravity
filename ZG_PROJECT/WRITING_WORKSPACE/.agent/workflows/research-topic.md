# Research Topic Workflow

1. Ask user for the "Topic" or "Keywords".
2. **Web Search**:
   - Search for the topic on Google.
   - Read top 3-5 relevant articles to understand the current landscape.
3. **Summarize**:
   - List key points found in the research.
   - Identify common questions (FAQs) related to the topic.
4. **Draft Outline**:
   - Propose an article structure (H2, H3 headings).
   - Introduction plan (Hook).
   - Conclusion plan (CTA).
5. **User Review**:
   - Present the outline to the user for approval.
   - If approved, proceed to `/create-draft` (optional).
