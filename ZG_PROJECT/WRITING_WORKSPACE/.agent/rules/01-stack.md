# 技術スタック (Writing Workspace)

## Language & Format
- **Markdown**: GFM (GitHub Flavored Markdown)
- **Language**: Japanese (Primary), English (Secondary)

## Linting & Quality
- **textlint**: 日本語の文章校正
  - `textlint-rule-preset-ja-spacing`
  - `textlint-rule-preset-ja-technical-writing`
- **MarkdownLint**: マークダウン記法の統一

## Tools (Optional)
- **Marp**: スライド作成が必要な場合
- **Hugo**: 静的サイト生成が必要な場合
- **Pandoc**: フォーマット変換（docx, pdf等）

## Version Control
- **Git**: 記事の履歴管理
