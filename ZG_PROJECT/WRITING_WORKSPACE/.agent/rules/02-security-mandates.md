# セキュリティ義務 (Writing Workspace)

## プライバシーと機密情報の保護

### 1. 下書きの取り扱い (Draft Privacy)
- 公開前の下書き（Draft）を、許可なく外部のパブリックな場所にアップロードしてはならない。
- `public` フォルダや公開リポジトリにコミットする際は、必ず `draft: false` フラグを確認すること。

### 2. 個人情報の保護 (PII)
- 記事内に個人情報（メールアドレス、電話番号、住所）が含まれていないか確認すること。
- スクリーンショットを掲載する場合は、機密情報や個人情報部分をマスキングすること。

### 3. ソースの検証
- 引用元や参照元のURLが安全であることを確認する（フィッシングサイト等でないこと）。
